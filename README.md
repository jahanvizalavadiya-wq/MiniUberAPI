Mini Uber Clone is a full-stack web application demo that simulates basic ride-booking functionality similar to Uber. This project uses ASP.NET Core Web API as the backend and React.js with Leaflet Maps as the frontend.

The application allows users to enter pickup and drop locations and display:

Available drivers

Route visualization

Distance calculation

Estimated Time of Arrival (ETA)

This version uses static data without database integration.
